# MINESWEEPER
I wanted to develop a minesweeper game in vanilla JS,
however I really appreciate the typing of typescript.
So I decided to use typescript and compile it to js.

# LANGUAGES
- Typescript
- HTML
- CSS

# Dependencies
- npm

# How to use
- Clone the repo and run `npm install` to install the dependencies.
- Run npm run
