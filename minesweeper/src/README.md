# Minesweeper

Vanilla js (no framework, only ts) minesweeper
app.

Art made by me

## Folder dist
### back
The backend of the game. Most of the logic is controlled here. Since it's a simple game I'll keep both front and back together.

### front
The visual part of the project. Since I have one of the ugliest assets I've ever done (made in like 5 min), I'll try to improve the front part.


The vanilla js decision was to assure the project remains playable for as long as possible. Since most frameworks get deprecated pretty easy (and sometimes they have like pretty serious vulnerabilities) I decided to KISS.
